Editor’s Note

Many readers and contributors have expressed an interest in Shenandoah‘s plans for the future.  The application deadline for the position of Shenandoah editorship has passed.  The Washington and Lee English Department’s search committee is currently reviewing and assessing the applications.

We hope to be able to announce the new editor upon the release of next spring’s issue and to feature a brief interview addressing her/his hopes and expectations for the future.

Layover in Town by Gordon Phillips (American, 1927-2011)
